Як правильно встановити python?


Крок 1: Завантажте Python

Перейдіть на офіційний веб-сайт Python за адресою https://www.python.org/. На головній сторінці ви побачите кнопку "Downloads". Клацніть на неї.


Крок 2: Виберіть версію Python

На сторінці завантаження вам буде пропоновано вибрати версію Python. Рекомендовано завантажити останню стабільну версію (на момент моєї останньої інформації це була версія 3.x.x).


Крок 3: Виберіть платформу

Оберіть вашу операційну систему. Якщо ви використовуєте Windows, вам може бути запропоновано вибрати між 32-бітною та 64-бітною версією. Виберіть ту, яка підходить для вашої системи.


Крок 4: Запустіть інсталятор

Завантажений файл - це інсталятор Python. Відкрийте його та слідуйте інструкціям інсталятора. Переконайтеся, що ви встановлюєте опцію "Add Python to PATH" (додати Python до системного шляху), це спростить вам викликати Python з командного рядка.


Крок 5: Перевірте встановлення

Відкрийте командний рядок (на Windows ви можете використовувати Command Prompt або PowerShell, на Unix-подібних системах - Terminal) і введіть команду:


python --version
